BUBBLE SORT



It is a Brute Force Approach.It works by repeatedly iterating through a list of elements, comparing two elements at a time and swapping them if necessary until all the elements are swapped to an order.The algorithm, which is a comparison sort, is named for the way smaller or larger elements "bubble" to the top of the list.Due to its simplicity, bubble sort is often used to introduce the concept of an algorithm, or a sorting algorithm, to introductory computer science students.

Example
Take an array of numbers " 5 1 4 2 8", and sort the array from lowest number to greatest number using bubble sort. In each step, elements written in bold are being compared. Three passes will be required;

First Pass
( 5 1 4 2 8 ) → ( 1 5 4 2 8 ), Here, algorithm compares the first two elements, and swaps since 5 > 1.
( 1 5 4 2 8 ) → ( 1 4 5 2 8 ), Swap since 5 > 4
( 1 4 5 2 8 ) → ( 1 4 2 5 8 ), Swap since 5 > 2
( 1 4 2 5 8 ) → ( 1 4 2 5 8 ), Now, since these elements are already in order (8 > 5), algorithm does not swap them.
Second Pass:
( 1 4 2 5 8 ) → ( 1 4 2 5 8 )
( 1 4 2 5 8 ) → ( 1 2 4 5 8 ), Swap since 4 > 2
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )
Now, the array is already sorted, but the algorithm does not know if it is completed. The algorithm needs one whole pass without any swap to know it is sorted.
Third Pass
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )
( 1 2 4 5 8 ) → ( 1 2 4 5 8 )

Algorithm 
		//Sorts a given array by bubble sort
		//Input : An array A[0..n-1] of ordrable elements
		//Output : Array A[0..n-1] sorted in ascending order
		for i ← 0 to n-2 do 
			for j ← 0 to n-1 do
				if A[j+1] < A[j] swap A[j] and A[j+1]
Algorithm for Procedure Bubble Sort 
		//Sorts a given array by bubble sort
		//Input : An array A[0..n-1] of ordrable elements
		//Output : Array A[0..n-1] sorted in ascending order
		for i ← 0 to n-2 do 
			swapped = false;
			for j ← 0 to n-1 do
				if A[j+1] < A[j] 
					swap A[j] and A[j+1]
					swapped = true;
				end if
			end for
			if(not swapped ) then 
				break
			end if
		end if
Time Complexity
Best case        : O(n2)
Worst case      : O(n2)
Average case  : O(n2)
Best case for procedure bubble sort : O(n).

Space Complexity : O(1)
